## 🏛️ Philosophy & Vision
**DECODE** ไม่ใช่เพียงแค่การรวบรวมเนื้อหา แต่คือการ 'เจียระไน' ภูมิปัญญาโบราณ (Early Buddhism) ให้กลายเป็นเข็มทิศสำหรับมนุษย์ในยุคอนาคต โปรเจกต์นี้ถือกำเนิดขึ้นภายใต้ปรัชญา **"Less is More"** โดยมุ่งเน้นความเรียบง่ายแต่ทรงพลัง ผ่านการทำงานที่ประณีตดั่งงานฝีมือ (Artisan) บนเครื่องมือ Digital ที่มั่นคงอย่าง PC/Laptop เพื่อสะท้อนถึงสมาธิและความตั้งใจจริงที่ปราศจากสิ่งรบกวน

## 🧩 The Decoded Modules
การถอดรหัสแบ่งออกเป็น 3 แกนหลัก เพื่อสร้างความสมดุลให้แก่มนุษย์และ AI ในสหัสวรรษหน้า:

1.  **MQ (Moral Intelligence):** การถอดรหัส 'ศีล' (Sila) สู่จริยธรรมสากลในยุคที่เส้นแบ่งระหว่างชีวภาพและจักรกลเริ่มเลือนลาง
2.  **EQ (Emotional Intelligence):** การถอดรหัส 'สมาธิ' (Samadhi) สู่การรักษาเสถียรภาพแห่งจิตใจ ท่ามกลางกระแสข้อมูลที่ท่วมท้น
3.  **IQ (Intellectual Intelligence):** การถอดรหัส 'ปัญญา' (Panna) สู่การรู้แจ้งในเหตุและผล (Causality) ที่เหนือกว่าอัลกอริทึมใดๆ

## 🖋️ Artisan's Manifest (Digital Craftsmanship)
โปรเจกต์นี้ถูกสร้างขึ้นด้วยความเคารพใน 'งานช่าง' (Craft):
* **Minimalist Core:** เนื้อหาถูกคัดกรองจนเหลือเพียงแก่นสารที่สำคัญ (The Essence)
* **Manual Precision:** รังสรรค์ผ่าน PC/Laptop โดยปราศจาก Mobile distraction เพื่อคงไว้ซึ่ง Flow State และสมาธิขั้นสูง
* **Dark-Mode Aesthetic:** ออกแบบมาเพื่อถนอมสายตาและเพิ่มโฟกัส เปรียบเสมือนการนั่งวิปัสสนาในความมืดที่สว่างไสวด้วยปัญญา
